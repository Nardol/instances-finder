# Instances Finder <VERSION>

Highlights

- Accessible results list with desktop-like keyboard navigation.
- Tauri host hardened with a minimal clipboard allowlist and strict CSP.
- Linux AppImage and Debian package.
- Windows portable archive and NSIS installer.

Security & Privacy

- The Instances.social token is stored in the OS keyring only when requested and is never logged.
- Application network access remains restricted to instances.social.

Thanks for trying the prototype. Accessibility and performance feedback is welcome.
